How to export from CDR.fyi
  1. Open cdr.fyi → Order Explorer
  2. Apply any filters you need (year, method, status…)
  3. Click Export CSV (top-right of the table)
  4. Upload the downloaded .csv file here — it will be auto-detected
Smart deduplication: records already in the database are automatically skipped. Only new orders are added.
