La sync incrementale scarica solo gli ultimi 30 giorni. La sync completa riscansiona tutto il dataset.
Configura Token CDR.fyi
Ottieni il token da: portal.cdr.fyi → Settings → API Access → Generate New Token
Come funziona la sync automatica
Ogni lunedì alle 04:00 il server scarica automaticamente gli ultimi 30 giorni dall'API CDR.fyi
I duplicati vengono ignorati automaticamente
Puoi eseguire la sync manuale in qualsiasi momento
Senza token: usa "Import Data" per caricare il CSV
API: cdr.fyi/resources/api-docs
Aggiornamento manuale (senza token)
Se non hai il token API, scarica il CSV dal portale CDR.fyi e caricalo qui:
  1. Vai su portal.cdr.fyi
  2. Accedi al tuo account
  3. Scarica il CSV delle transazioni (Export / Download)
  4. Vai su Import Data nella sidebar
  5. Carica il file CSV — i duplicati vengono rilevati automaticamente
