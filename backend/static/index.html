Colonne Riconosciute

Supplier: supplier, supplier name, project name

Acquirente: purchaser, buyer, company
Tonnellate: tonnes, volume, tco2e, amount

Regione: region, continent, geography

Aggiornamento Automatico 24/7

Per automatizzare l'importazione senza intervento manuale:

  1. Avvia lo scheduler: python utils/scheduler.py
  2. Copia i nuovi file nella cartella: backend/data/inbox/
  3. Lo scheduler controlla ogni 15 minuti e importa automaticamente
  4. I file elaborati vengono spostati in data/processed/
